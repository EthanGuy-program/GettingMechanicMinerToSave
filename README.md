# GettingMechanicMinerToSave
This is the powershell program to get the steam version of Mechanic Miner by Hello John to properly save

Instructions:
Simply replace "Storytime" in "$base\games\Storytime" with the save game name
Run the program with % "DIRECTORY/watch_save.ps1" amd replace DIRECTORY with where the ps1 is

NOTICE:
Roaming data must be in the drive that the powershell program is run in.
Check "C:\Users\USER\AppData\Roaming\mechanicminer" while replacing USER with the USER you installed mechanic miner on.
